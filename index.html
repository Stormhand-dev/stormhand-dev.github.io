‚öôÔ∏è Important Installation Notes

All modules are optional and independent ‚Äî with a few important exceptions:

  • PATCHES B, D, and E are parts of one large Environment and World Patch. They must be installed together, or not at all, in order to work properly.
  • PATCH-L requires PATCH-A to be installed (mandatory).
  • PATCH-U requires PATCH-A and PATCH-G to be installed (mandatory).
  • VanillaHelpers is mandatory for the project to operate properly.
  • DXVK is strongly recommended ‚Äî and effectively required. It provides a modern Vulkan-based rendering path that greatly improves stability, memory handling, and reliability when using high-resolution textures.

Installation Guide

  1. Install required dependencies (DXVK + Vanilla Helpers).
  2. Download the desired patch modules and place them inside your Data folder.
  3. Install Reshade (optional)
  4. Copy my personal Reshade settings and DXVK configuration file into your World of Warcraft main folder.
  5. Launch the game.
