Installation Guide

  1. Install required dependencies (DXVK + Vanilla Helpers).
  2. Download the desired patch modules and place them inside your Data folder.
  3. Install Reshade (optional)
  4. Copy my personal Reshade settings and DXVK configuration file into your World of Warcraft main folder.
  5. Launch the game.
